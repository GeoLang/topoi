Features

📊 Geometry Types

  • Point, Coord (2D)
  • LineString (open polyline)
  • Ring (closed, oriented)
  • Polygon (exterior + holes)
  • MultiPolygon (collection)
  • Envelope (AABB)

🔍 Spatial Predicates

  • Point-in-polygon (ray casting)
  • Polygon-polygon envelope intersection
  • Contains with hole exclusion
  • Envelope containment
  • Ring winding detection (CCW/CW)

📏 Measurements

  • Polygon area (Shoelace formula)
  • Signed area (winding-aware)
  • LineString length
  • Polygon centroid
  • Point-to-point distance

🛡️ Operations

  • Polygon buffering (vertex offset)
  • Envelope union & intersection test
  • Ring closure validation
  • Convex polygon support

Quick Start

use topoi_core::{Coord, Polygon, Ring, contains};

let ring = Ring::new(vec![
    Coord::new(0.0, 0.0), Coord::new(4.0, 0.0),
    Coord::new(4.0, 4.0), Coord::new(0.0, 4.0),
    Coord::new(0.0, 0.0),
]);
let polygon = Polygon::new(ring, vec![]);
assert!(contains(&polygon, &Coord::new(2.0, 2.0)));
